MAE 3780 Robot Project

Autonomous Cube-Collecting Robot

For the MAE 3780 Robot Project, my team designed, built, and programmed an autonomous robot to collect cubes placed in the center of a competition field. Our robot used two front doors that opened outward so cubes could enter underneath the robot as it drove forward. The door system was controlled by a servo motor, torsional springs, and a string mechanism. We also added side extensions to increase the amount of space available for cube collection.

I contributed to the mechanical design, testing, and integration of the robot’s cube collection system. I helped build and adjust the front-door mechanism, test the spring and servo behavior, and troubleshoot how the mechanical system worked with the robot’s wiring and code. This project taught me how important mechanical reliability is in robotics because even when the code works, small issues like loose mounting, slipping springs, or shifting components can affect the robot’s competition performance.
